Getting Started with OFEM

Follow these simple steps on a Mac to run OFEM. No programming knowledge required.

  1. Install Requirements
    1. Install Node.js (LTS version) and follow the installer prompts.
    2. Ensure the PostgreSQL App is running (it's already installed).
  2. Download and Unzip
    1. Download the project ZIP file.
    2. Open your Downloads folder and double‑click the ZIP to create the OFEM folder.
  3. Install Packages
    1. In the OFEM folder, double‑click install.command. It installs everything you need.
  4. Create the Database
    1. Click the button below. A wizard opens in Terminal, creates a database with a random name, user, and password, then updates your .env file automatically.
    3. Leave the Terminal window open until it shows “Database setup complete!”.
  5. Run the App
    1. Double‑click start.command to launch the server.
  6. Open the Dashboard
    1. When Terminal shows “OFEM server listening on http://localhost:3000”, open a web browser.
    2. Visit http://localhost:3000.

The app is now ready. Use the dashboard buttons to update fan names and send messages.
