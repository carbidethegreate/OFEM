Getting Started with OFEM

Follow these steps on a Mac to run OFEM for the first time. No programming knowledge required.

  1. Install Requirements
    1. Install Node.js (LTS version) and follow the installer prompts.
    2. Install PostgreSQL (or another PostgreSQL distribution) and start the database server.
  2. Download and Unzip
    1. Download the project ZIP file.
    2. Open your Downloads folder and double‑click the ZIP to create the OFEM folder.
  3. Create the Configuration File
    1. Open the OFEM folder and find .env.example.
    2. Make a copy of it named .env.
    3. Open .env with TextEdit and replace the placeholder values:
      • DATABASE_URL – your PostgreSQL connection string (for example postgres://user:password@localhost:5432/ofem).
      • ONLYFANS_API_KEY – your OnlyFans API key.
      • OPENAI_API_KEY – your OpenAI API key.
    4. Save the file.
  4. Prepare the Database
    1. If the database named in DATABASE_URL does not exist, create it using the PostgreSQL app or run createdb ofem in Terminal.
  5. Run the App
    1. In the OFEM folder, double‑click start.command. A Terminal window opens.
    2. The script installs needed packages (only the first time) and starts the server.
  6. Open the Dashboard
    1. When Terminal shows “OFEM server listening on http://localhost:3000”, open a web browser.
    2. Visit http://localhost:3000.

The app is now ready. Use the buttons in the dashboard to update fan names and send messages.
