# Project Setup

## Required Tools
- Node.js 14+
- PostgreSQL
- OnlyFans API key
- OpenAI API key

## Steps
1. Run `npm install`.
2. Configure environment variables via `./setup-env.command` or `node setup-env.js`.
3. Create the database using `./setup-db.command` or `node setup-db.js`.
4. Start the server with `npm start`.
